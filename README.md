### Smartwatch Text Entry Interface (CMU Graphics / Processing)

## üîç Project Overview

**WristCrafters** is a smartwatch text entry system designed and implemented using **Processing (CMU Graphics)**. The project explores how users can efficiently and accurately type text on a **small, constrained smartwatch screen**, focusing on interaction design, usability, and iterative improvement through user studies.

The system was developed through multiple prototypes, user testing rounds, and refinements, resulting in a final design that significantly improves **typing speed, accuracy, and user confidence** compared to the provided scaffold design.

---

## üéØ Project Goals

- Design a **fast and accurate smartwatch text entry method**
- Explore alternatives to traditional QWERTY keyboards on small screens
- Use **user-centered design** and iterative testing to guide improvements
- Quantitatively evaluate performance using **WPM and accuracy metrics**

---

## üß© Core Features

### Text Entry Interface
- Letters arranged around the smartwatch screen
- Clearly separated **SPACE** and **DELETE** buttons
- Grid-based layout to reduce mistaps
- Visual hover highlighting to indicate selection
- Sound feedback on key press

---

### Navigation & Editing
- Arrow buttons to move the text cursor
- Ability to correct mistakes without restarting input
- Clear visual indication of selected characters

---

### Autofill & Prediction (Final Prototype)
- Word suggestion (autofill) based on typed prefix
- Reduced number of required taps
- Improved typing speed and usability

---

## üß™ User Studies & Evaluation

The system was evaluated through **multiple rounds of user testing**, including think-aloud studies, surveys, and interviews.

### Quantitative Results
- **Scaffold design average:** ~7.32 WPM  
- **Final design average:** **~15.04 WPM**
- Significant improvement in both speed and accuracy

### Key Findings
- Hover highlighting increased user confidence
- Sound feedback improved typing rhythm
- Clearly labeled SPACE and DELETE buttons reduced confusion
- Autofill feature significantly improved typing speed
- Arrow keys enabled efficient error correction
